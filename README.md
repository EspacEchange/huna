* [Python 2.7](https://www.python.org)
* [Flask](http://flask.pocoo.org)
* [New Relic API](http://new-relic-api.readthedocs.org/en/latest/index.html)
* [Cloudera API](http://cloudera.github.io/cm_api/docs/python-client/)
* [Jenkins API](https://jenkinsapi.readthedocs.org/en/latest/)
* [Google Analytics API](https://developers.google.com/api-client-library/python/apis/analytics/v3)
